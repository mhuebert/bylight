Bylight Example

Basic Usage

Here's an example of a function definition in JavaScript: 

function greet(name) {
    console.log(`Hello, ${name}!`);
}

greet("Alice");
greet("Bob");

Multiple Patterns

We can highlight if statements and else clauses in this example: 

function checkAge(age) {
    if (age >= 18) {
        console.log("You are an adult.");
    } else {
        console.log("You are a minor.");
    }
}

checkAge(20);
checkAge(15);

Regex Pattern

Here's an example using a regex pattern to match const declarations. Note that one must URL-encode the regex, so + is written %2B. 

const PI = 3.14159;
const MAX_SIZE = 100;
let variable = "This won't be highlighted";
const COLORS = ['red', 'green', 'blue'];

Directional Highlighting

Use the in query parameter to specify which pre's to match, relative to the link.

1 + 1
1 + 2
1 + 3

Highlight 1 in the first, second, and third pre's above.

Use the dir param to select all matches up or down from a link. Here we highlight all the 1's below.

1 + 4
1 + 5
1 + 6
